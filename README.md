# Installation

## Step 1: 

```git clone https://github.com/Dagurf21/T-113-VLN1.git```

## Step 2:

Fyrir module þarfir þarf bara að skrifa:

```pip install -r requirements.txt```

Inn í terminal-ið.

## Step 3:

Keyrðu forritið með:

```cd T-113- && python main.py```
